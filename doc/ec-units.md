# Epimethean Challenge — Unit List

## Natural Resources

| Code | Unit | Description |
|------|------|-------------|
| FUEL | Fuel | Used for all production and transportation |
| METL | Metallics | All metallic substances other than gold |
| NMTL | Non-Metallics | All non-metallic substances |

## Population Units

| Code | Unit | Description |
|------|------|-------------|
| PRO | Professionals | Those who require long apprenticeships or a great deal of training |
| SOL | Soldiers | All military personnel |
| UEM | Unemployables | Child-rearing and pregnant females, children, combat casualties, and the elderly |
| USK | Unskilled | Those who do not require long apprenticeships or a great deal of training |

## Cadre

| Code | Unit | Description |
|------|------|-------------|
| CNST | Construction | Execute assembly and dis-assembly orders |
| RBL | Rebels | Tally of population willing to rebel (not an actual class) |
| SPY | Spies | Report on other players and incite rebellion |
| TRNE | Trainees | Unskilled workers in training |
| WRKR | Workers | Professional and unskilled workers allocated to a Factory, Farm, or Mine |

## Weapons

| Code | Unit | Description |
|------|------|-------------|
| AMSL | Anti-Missiles | Launched by missile launchers to destroy attacking missiles |
| ACFT | Assault Craft | Land/space vehicles used to invade colonies or ships |
| AWPN | Assault Weapons | Used by soldiers on the surface of a planet |
| ESHD | Energy Shields | Deflect energy beams |
| EWPN | Energy Weapons | Project a powerful beam of concentrated energy; used in all combat except surface colony to surface colony |
| MRBT | Military Robots | Can replace soldier units (TL × 2 soldiers per unit) |
| MSUP | Military Supplies | Ammunition, medicines, etc., used up during combat (1 unit per soldier per combat round) |
| MSSL | Missiles | Used in any kind of combat; less accurate than energy weapons |
| MLNC | Missile Launchers | Launch missiles; accuracy depends on TL of the launcher |

## Production Units

| Code | Unit | Description |
|------|------|-------------|
| FACT | Factories | Manufacture all units except natural resources, food, and population |
| FARM | Farms | Produce food rations (TL 1: open-colony farm; TL 2–5: hydroponic; TL 6–10: artificial-sunlight hydroponic) |
| MINE | Mines | Extract natural resources from deposits |

## Research

| Code | Unit | Description |
|------|------|-------------|
| LAB | Laboratories | Production unit that generates research points |
| RP | Research Points | Used to advance tech levels |
| PRTO | Prototypes | Used to transport tech levels |

## Miscellaneous / Structural

| Code | Unit | Description |
|------|------|-------------|
| AUTO | Automation | Replace unskilled workers in factories, farms, or mines (unit × TL = worker units replaced) |
| CSGD | Consumer Goods | Produced by factories; used to pay the population |
| FOOD | Food | Consumed by all population units (¼ food unit per population unit per turn) |
| HDRV | Hyper Engines | Propel ships through hyper-space; jump range = 1 light-year × TL |
| LSU  | Life Support | Recycle air and water in ships and enclosed colonies (supports TL² population units) |
| PROB | Probes | Executed by sensors; report on ships, colonies, and resource deposits in the same system |
| SENS | Sensors | Detect ships and colonies in orbit; conduct probes (1 × TL probes per turn) |
| SDRV | Space Drives | Maintain orbit and maneuver in combat (anti-gravity; cannot be used for interplanetary/interstellar travel) |
| STRC | Structural | Required to build ships and colonies; see technical documentation for STRC-2 (light structural) variant |
| TRNS | Transports | Transfer population and materials between ships/colonies at the same planet |
